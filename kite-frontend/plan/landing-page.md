# Landing Page Planning

## 1. 개요
루트 경로(`/`)로 접속했을 때 가장 먼저 노출되는 서비스의 대문(Landing) 역할을 합니다.

## 2. 라우트 경로
- `/`: 랜딩 페이지

## 3. 주요 화면 및 기능 스펙

### A. 히어로 섹션 (Hero Section)
- **Kite 프로젝트 소개**: "Kubernetes 기반의 안전하고 격리된 가상머신 프로비저닝 플랫폼" 등 간결하고 명확한 카피라이팅 적용.
- **CTA (Call to Action) 버튼**: `[로그인 및 시작하기]` 버튼을 배치하여 사용자를 `/login` 화면으로 자연스럽게 유도.

### B. 주요 기능 요약 (Features)
- 텍스트와 간단한 아이콘(Ant Design Icons 활용)을 사용하여 Kite가 제공하는 기능을 2~3개 정도로 요약해서 설명.
  - 예: 즉각적인 VM 할당, 네임스페이스 기반의 완벽한 보안 격리, 직관적인 대시보드 제어 등.

### C. UI 레이아웃
- 로그인 상태일 경우 루트(`/`)에 접속하면 자동으로 `/dashboard` 또는 `/admin`으로 리다이렉트 되도록 Auth Guard 연계 처리.
- 미로그인 사용자에게만 화면 전체를 활용하여 화사하고 쾌적한(Bright/Cream 테마) 첫인상을 제공.
